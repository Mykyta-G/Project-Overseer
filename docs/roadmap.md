# Overseer Roadmap

From zero to Jarvis, one step at a time.

## Phase 1 — Foundation *(Day 1-2)*
The core that everything else plugs into.

- [x] **Step 1: Telegram Bot** — Overseer responds via Telegram through Clawdbot
- [x] **Step 2: Agent Status Queries** — "status" returns fleet overview
- [x] **Step 3: Priority Filter** — classify all info as 🔴/🟡/🟢/⚪
- [ ] **Step 4: Model Fallback Chain** — Haiku → Gemini Flash → Forge

## Phase 2 — Daily Intelligence *(Day 3-5)*
The flows that make Overseer useful every day.

- [ ] **Step 5: Morning Brief** — 08:30 daily: agent activity, calendar, weather
- [ ] **Step 6: Evening Summary** — 21:00 daily: done, open, tomorrow
- [ ] **Step 7: Discord Filter** — monitor Discord, only forward what matters
- [ ] **Step 8: Email Triage** — scan inbox, classify, draft replies

## Phase 3 — Location Intelligence *(Day 5-7)*
Jarvis knows where you are.

- [ ] **Step 9: iPhone Location Bridge** — iOS Shortcut → Pi webhook on arrive/leave work
- [ ] **Step 10: Transit Integration** — check departure times, nudge to go home

## Phase 4 — Knowledge & Memory *(Week 2)*
Overseer knows your plans, goals, and context.

- [ ] **Step 11: Notion Integration** — read goals, write daily summaries
- [ ] **Step 12: WhatsApp Standup** — monitor team chat, summarize standups

## Phase 5 — Local AI *(Week 2-3)*
Free, private, always available.

- [ ] **Step 13: Wake-on-LAN** — wake Gaming PC for Forge on demand
- [ ] **Step 14: Route to Forge** — offload routine tasks to local Qwen 3

## Phase 6 — Physical Interface *(When Tabbie ships)*
The Jarvis layer.

- [ ] **Step 15: Tabbie Desk Robot** — physical display on monitor
- [ ] **Step 16: Wake Word** — "Overseer" for hands-free access
- [ ] **Step 17: Agent Voice Profiles** — each agent gets unique voice
- [ ] **Step 18: Agent Avatars** — animated faces per agent
- [ ] **Step 19: Mission Control Dashboard** — live agent status on Tabbie screen

## Phase 7 — Screen Awareness *(Month 2-3)*
Point at something and Overseer knows what it is.

- [ ] **Step 20: Screen + Mouse Context** — screenshot + cursor region → Claude Vision
- [ ] **Step 21: Live Build Narration** — watch agents code in real-time via Tabbie

### The One Rule
Build one thing at a time. Get it solid. Then add the next.
